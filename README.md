## :page_facing_up: How to run

Clone this repository:

```sh
  $ git clone https://github.com/luc-ribeiro/client-list-flask.git
```
Install dependencies:

```sh
  $ pip install -r requirements.txt
```

Run the command 

```sh
  $ python3 app.py
```

Access the project at `localhost:5000`
